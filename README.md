# CRM v2.0

A modern Customer Relationship Management system built with cutting-edge technologies.

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Features

- Customer management
- Sales pipeline tracking
- Contact management
- Reporting and analytics
